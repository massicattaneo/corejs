COREJS - v1.0.0
===============

Basic Javascript Framework: 

- String extensions
- Array extensions
- Object extensions
- HTML Element extensions
- Collections
- Promises
- Dependecy Injection
- Http requests

UNIT TEST
---------
```
npm run ut
```

Or to run tests in a single folder
```
npm run ut -- --test=<fodler-name>
```

BUILD
-----
```
npm run dist
```
